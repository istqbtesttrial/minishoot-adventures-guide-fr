# Notes de sources

Ce repo s’appuie principalement sur :

- la carte interactive communautaire
- le guide illustré GitHub
- les discussions Steam sur les objets/modules manquants
- des retours Reddit pour les cas particuliers

Le but n’est pas de recopier les contenus externes, mais de synthétiser les informations les plus utiles en français.
