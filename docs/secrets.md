# 🧱 Secrets et objets faciles à rater

> Les infos importantes issues des guides, discussions et données carte.
>
> 🗺️ Quand un secret correspond à un objet placé, la carte interactive fait foi. Cette page explique surtout les conditions, interactions et accès cachés.

---

## 🏠 Family Home / Trouver sa place

Source YAL : le succès **Finding your place / Trouver sa place** s’obtient en entrant dans un bâtiment et en s’asseyant sur une chaise après avoir sauvé les trois vaisseaux / membres de la famille.

À faire :

- sauver les proches / amis requis ;
- retourner à la maison familiale ;
- entrer dans le bâtiment ;
- se placer / s’asseoir sur la chaise ;
- vérifier le succès et les récompenses associées.

Pourquoi c’est important : plusieurs joueurs pensent avoir 100% carte, mais il manque encore une récompense liée à la famille.

---

## ❤️ Wounded Heart / Cœur Blessé

Repères confirmés par les sources :

- sud-ouest de la carte ;
- bâtiment type chapelle / temple ;
- statue ou autel suspect ;
- passage caché à droite / derrière l’autel.

À vérifier aussi en post-game si une route / clé semble manquer.

---

## 🏁 Tour aux 3 slimes / Course #2

La course #2 est l’un des accès les plus cachés.

À retenir :

- tour avec symbole de **3 slimes** ;
- 3 slimes / cercles à déclencher dans la zone ;
- un token / bouton peut déplacer un rocher ;
- accès vers une zone souterraine ;
- piste de course à l’est de cette zone ;
- lien avec une grosse clé / section underground.

Voir : [Recherche deep — Course #2](recherche-courses-deep.md#course-2--acces-detaille-car-cest-la-plus-problematique)

---

## 🎁 Récompense finale des 8 courses

Les 8 courses donnent un objet utile.

À retenir :

- gagner les courses ne suffit pas ;
- récupérer la récompense finale après les 8 courses ;
- cet objet sert à entrer dans un bonus dungeon post-game ;
- sans lui, les extras peuvent rester bloqués.

---

## 🏜️ Donjon optionnel du désert

Après le Donjon 3, un détour vers le désert à l’ouest est recommandé.

Contient :

- un pouvoir / item utile ;
- un scarabée ;
- un boss / succès lié au désert.

---

## 🏚️ Bâtiment caché de la région ouest

Source YAL : dans la région du Donjon 4, un bâtiment accessible après un petit travail contient :

- un pouvoir ;
- un bouton bien caché ;
- un scarabée bien caché.

Les égouts / souterrains de cette région contiennent aussi plusieurs objets.

---

## 🪲 Scarabée underground sud-ouest

Source YAL : un scarabée se trouve dans une zone souterraine accessible via un bâtiment au sud-ouest. L’accès semble demander Surf + PowerShot.

À vérifier si :

- il manque un scarabée ;
- une zone souterraine paraît inaccessible ;
- la carte montre un point qui semble mal placé.

---

## 🕳️ Bons trous / chutes utiles

Le guide YAL signale plusieurs endroits accessibles en tombant dans le bon trou.

À retenir :

- certaines chutes sont volontaires ;
- revisite les trous après nouveaux pouvoirs ;
- certains succès / objets peuvent dépendre de ces accès.

---

## 🎵 Barde / Merci de m’avoir écouté

Source YAL : après avoir sauvé le barde, reste près de lui à la sortie nord de la ville pendant un moment pour obtenir **Merci de m’avoir écouté**.

---

## 🐢 Tortue / arbre / vraie route

Sources succès :

- **Vieil Ami** est lié à la tortue géante ;
- **Fini de ronfler** est lié à l’arbre et au barde ;
- ces éléments sont à vérifier en post-game / nettoyage final.

---

## ⚠️ Sans regarder en arrière

Source YAL : l’option finale du bonus dungeon peut oblitérer / supprimer la sauvegarde.

À faire uniquement quand :

- modules faits ;
- scarabées faits ;
- courses + récompense faites ;
- lore fait ;
- arènes faites ;
- succès cachés faits.

---

## Résumé des plus gros oublis

- Family Home / chaise
- Wounded Heart / autel
- course #2 / tour aux 3 slimes
- récompense finale des courses
- donjon optionnel désert
- bâtiment caché zone ouest
- égouts / souterrains
- scarabée underground sud-ouest
- barde à la sortie nord
- option finale “Sans regarder en arrière” à garder pour la toute fin
