## Informations générales importantes

- Le guide YAL précise qu’il ne couvre pas tout ce que le jeu permet de localiser clairement avec ses propres méthodes : il vise surtout les lieux piégeux.
- Les succès de difficulté dépendent de la **plus basse difficulté utilisée sur la sauvegarde**.
- Le mode de visée / aiming mode n’affecte pas les succès.
- Les courses donnent un objet utile avant la fin et un accès important aux extras après le boss final.
- La carte interactive est la **référence absolue** pour les objets placés, coordonnées, catégories et quantités.
- Les guides, discussions et vidéos servent à expliquer le chemin, les conditions, les retours PNJ, les récompenses différées et les pièges, mais ne doivent pas contredire la carte interactive.

## Donjon 1 — informations source

D’après le guide YAL :

- le premier pouvoir important est **Boost** ;
- l’entrée du donjon s’ouvre via un bouton ;
- le donjon introductif est assez simple ;
- il contient seulement quelques objets pas trop cachés.

À intégrer dans le guide joueur :

- ne pas forcer le 100% au Donjon 1 ;
- revenir ensuite tester les petits gaps autour du village ;
- noter les objets visibles mais pas encore accessibles.

## Donjon 2 — informations source

D’après le guide YAL :

- pour y aller, il faut partir au sud vers les bois, faire un crochet vers l’est et trouver un bâtiment ;
- ce bâtiment mène vers une zone à l’ouest de l’entrée initiale ;
- ensuite, il faut remonter au nord pour atteindre la zone du donjon ;
- un autre bâtiment optionnel dans la zone contient du contenu utile ;
- pour rejoindre l’objectif, il faut entrer dans un trou ;
- battre une grosse tourelle ouvre un pont au sud ;
- en passant par le coin sud-ouest de la carte, il faut vérifier une statue suspecte dans un bâtiment type chapelle ;
- dans le donjon : scarabée au nord-est derrière mur caché ; fragment de cœur au nord-ouest ;
- l’upgrade du donjon ouvre plusieurs nouveaux accès sur la carte.

À intégrer :

- Donjon 2 = moment important pour commencer les vrais retours ;
- Wounded Heart / Cœur Blessé est lié au sud-ouest et à un autel / passage caché ;
- ne pas oublier le bâtiment optionnel.

## Donjon 3 — informations source

D’après le guide YAL :

- il faut aller dans la zone est ;
- puis monter au nord ;
- passer par un tunnel précis ;
- il y a une boutique : acheter la carte avant de continuer est conseillé ;
- plusieurs braziers / objectifs doivent être allumés ;
- une batterie est facilement oubliée : elle est derrière un mur caché près d’une piste de time trial, au nord-ouest du piédestal du pouvoir de zone ;
- un fragment de cœur à l’est s’ouvre via un bouton accessible par des tunnels depuis la partie centrale du donjon ;
- il y a deux scarabées : nord-est et ouest ;
- après le donjon, un détour vers le donjon optionnel du désert à l’ouest est recommandé ;
- ce donjon optionnel contient un pouvoir et un scarabée.

À intégrer :

- après Donjon 3, nettoyer intelligemment les tunnels / time trials / désert optionnel ;
- ne pas oublier la batterie près de la piste de contre-la-montre.

## Donjon 4 — informations source

D’après le guide YAL :

- l’accès à la région se fait par l’entrée en haut de la région ouest ;
- il y a un bâtiment dans lequel on peut entrer après un petit puzzle / travail ;
- ce bâtiment contient un pouvoir, un bouton bien caché et un scarabée bien caché ;
- les égouts / souterrains servent à traverser la zone mais contiennent aussi plusieurs objets.

À intégrer :

- région ouest = zone à fouiller méthodiquement ;
- bâtiment caché + bouton caché + scarabée caché ;
- égouts à traiter comme zone de collecte, pas seulement comme passage.

## Other bits / secrets divers

D’après le guide YAL :

- un scarabée se trouve dans une zone souterraine accessible via le bâtiment sud-ouest ;
- cet accès semble nécessiter Surf + PowerShot ;
- plusieurs lieux sont accessibles en tombant dans le bon trou ;
- le succès **Finding your place / Trouver sa place** s’obtient en entrant dans un bâtiment puis en s’asseyant sur une chaise après avoir sauvé les trois vaisseaux / membres de la famille ;
- le succès **Thanks for listening / Merci de m’avoir écouté** s’obtient en restant près du barde à la sortie nord de la ville après l’avoir sauvé ;
- une partie de l’underground avec une grosse clé est liée à la tour décrite dans la section course #2.

À intégrer :

- Family Home + chaise / siège ;
- barde à la sortie nord ;
- trous qui mènent à des zones cachées ;
- bâtiment sud-ouest / underground / scarabée ;
- lien entre tour aux 3 slimes et grande clé / souterrain.

## Finale

D’après le guide YAL :

- pour entrer dans le donjon final, il faut terminer deux mini-donjons ;
- l’un est ouvert ;
- l’autre s’ouvre avec un bouton sous un tas de gravats ;
- après les deux mini-donjons, on obtient un item qui ouvre le donjon final ;
- avant le boss, il y a un checkpoint et une option pour sortir ;
- si tu n’es pas assez fort, tu peux essayer puis revenir plus tard sans tout refaire.

À intégrer :

- pas besoin d’être 100% avant d’entrer dans le donjon final ;
- vérifier les upgrades si difficulté Avancée ;
- profiter du checkpoint / sortie avant boss.

---

## Extras / vraie fin

D’après le guide YAL :

- après avoir battu le jeu, il existe un bonus dungeon ;
- il faut l’objet obtenu après les 8 courses pour entrer ;
- après ce bonus, il existe encore un autre bonus dungeon ;
- ce second bonus contient des challenges et des boss fights revisités ;
- dans le même donjon, on trouve l’option de fin formelle qui **détruit / oblitère la sauvegarde** et donne le succès **Without looking back / Sans regarder en arrière**.

À intégrer :

- faire Sans regarder en arrière uniquement tout à la fin ;
- finir courses + reward avant les extras ;
- prévenir clairement du risque de suppression de sauvegarde.

## Succès / trophées FR importants

NationHive liste **34 succès**.

Succès critiques pour le 100% :

- **Suréquipément** — trouver tous les modules
- **Trouver sa place** — réunir la famille
- **Cœur Blessé** — trouver la source de la corruption
- **Plus rapide que vous tous !** — gagner contre tous les esprits
- **Ô Grand Primordial** — trouver tous les scarabées dorés
- **Passion pour l’Histoire** — lire toutes les anciennes tablettes
- **Pour la Gloire ?** — finir tous les niveaux de l’arène
- **C’est fini maintenant** — finir chaque niveau d’arène avec au moins 5 cœurs restants
- **Sans regarder en arrière** — abandonner les pouvoirs et quitter ce monde
- **Une épopée épique se termine** — finir en difficulté Avancée

Succès faciles à oublier :

- **Merci de m’avoir écouté** — rester près du barde
- **Quelle indignité !** — lié aux jarres / maison familiale selon retours
- **Quoique...** — démonter une amélioration
- **Je vole même pas ?!** — tomber dans l’eau
- **Aïe** — survivre à une chute importante
- **Pas à moi** — secret dans le secret dans la forêt
- **Vieil Ami** — réveiller la tortue géante
- **Fini de ronfler** — réveiller l’arbre grâce au barde

## Modules / objets manquants — retours joueurs

Les discussions Steam et Reddit confirment plusieurs pièges :

- module au-dessus / autour de Wounded Heart : souvent lié à Family Home ;
- module entre Enchanted Heart et Advanced Energy : Family Home ou achat scarabée oublié ;
- module malgré carte 100% : retourner dans la maison familiale, bâtiment bas-gauche de la zone de départ ;
- interaction précise : se placer / s’asseoir sur le siège après avoir réuni la famille ;
- Wounded Heart : temple sud-ouest, autel, passage secret à droite / derrière l’autel ;
- scarabée / underground sud-ouest : accès via bâtiment sud-ouest, possiblement après Surf + PowerShot ;
- dernière course : utiliser les checkmarks / lieux non validés.

## Build / confort

- Move Speed est très utile pour les courses.
- GodIsAGeek indique que Move Speed coûte `1 + 3 + 4`, donc **8 cristaux de niveau** pour le max.
- Réallouer temporairement les points vers mouvement / boost est conseillé pour les courses.
- Les options d’accessibilité peuvent réduire la frustration si nécessaire.
- Le mode de visée n’affecte pas les succès.
