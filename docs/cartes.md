## 📸 Guide visuel YAL / YellowAfterlife

<https://github.com/YAL-Game-Things/Minishoot-Adventures-Guide>

C’est la meilleure source visuelle trouvée. Elle contient :

- Donjon 1 : Boost, bouton d’entrée, objets simples ;
- Donjon 2 : route sud/bois/est, bâtiment optionnel, trou, tourelle, pont sud, chapelle / Wounded Heart, scarabée caché, cœur ;
- Donjon 3 : route est/nord/tunnel, carte à acheter, braziers, batterie cachée près time trial, cœur à l’est, scarabées, donjon désert ;
- Donjon 4 : région ouest, bâtiment caché, pouvoir, bouton caché, scarabée caché, égouts ;
- Courses : 8 courses + récompense finale ;
- Other bits : Family Home / chaise, barde, trous, scarabée underground ;
- Finale : deux mini-donjons, bouton sous gravats, item d’ouverture, checkpoint avant boss ;
- Extras : bonus dungeon, second bonus dungeon, Sans regarder en arrière.

Synthèse locale : [Recherche deep complète](recherche-deep-complete.md)
