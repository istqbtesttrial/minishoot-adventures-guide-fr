## 🎯 Convention d’annotation proposée

Pour rester lisible :

- cercle rouge = entrée / passage
- flèche jaune = chemin à suivre
- carré bleu = PNJ / récompense
- texte court en FR, jamais plus de 5 mots
- nom de fichier : `zone-sujet-etape.png`, ex. `family-home-siege.png`
