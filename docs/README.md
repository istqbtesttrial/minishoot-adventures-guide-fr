## 🗺️ Je cherche un emplacement exact

Ordre recommandé :

1. 🗺️ ouvrir la carte interactive : <https://minishoot-map.github.io/>
2. 🎚️ activer un seul filtre
3. 📊 vérifier les coordonnées dans [Extraction map](map-data.md)
4. 🎬 si besoin, regarder les vidéos listées dans [Cartes](cartes.md)
5. 📸 ajouter une future capture dans [Captures](captures.md)

Filtres les plus utiles :

- 🧩 **Modules & Skills**
- 🪲 **Scarabs**
- 👻 **Race spirits**
- ❤️ **Heart crystals**
- ⚡ **Energy upgrades**
- 🗺️📜 **Map & Lore fragments**
