## Méthode fiable

1. Ouvre <https://minishoot-map.github.io/>.
2. Active uniquement le filtre **Scarabs**.
3. Vérifie les grottes et zones souterraines autour du dernier marqueur.
4. Revisite les murs cachés si le point semble inaccessible.
5. Après les avoir tous trouvés, retourne au vendeur / robot lié aux scarabées pour récupérer les récompenses.
