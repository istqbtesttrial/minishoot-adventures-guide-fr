## Pour un 100% rapide

Si ton but est surtout de **trouver tous les modules** :

1. Ouvre la carte interactive : <https://minishoot-map.github.io/>
2. Active le filtre **Modules & Skills**
3. Vérifie en priorité :
   - la **Family Home**
   - la **petite grotte au sud-ouest** après avoir réuni la famille
   - **Wounded Heart** dans le temple au sud-ouest
   - la **récompense des 8 courses**
   - la **récompense des scarabées dorés**
